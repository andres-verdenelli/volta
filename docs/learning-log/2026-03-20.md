# Registro de aprendizaje

**Fecha:** 2026-03-20

## Estado actual

Hoy quedó armada una base inicial de la app:

- Layout global en Next con `app/layout.tsx`
- Header con navegación básica
- Home simple
- Página de login inicial
- Shadcn ya integrado para UI base

La app todavía está en una etapa muy temprana, pero eso es bueno: la estructura sigue chica, entendible y fácil de iterar.

## Stack decidido

- `Next.js`
- `shadcn/ui`
- `Drizzle ORM`
- `PostgreSQL` en `Supabase`
- Más adelante: `Clerk`
- Más adelante: `Zod`
- Más adelante: `React Hook Form`

## Criterio de avance

La idea no es aprender todo junto, sino ir agregando herramientas cuando aparezca una necesidad real.

Orden sugerido:

1. Entender bien rutas, páginas, layout y composición en `Next`
2. Resolver la UI base con `shadcn`
3. Armar flujo simple de pantallas
4. Recién después conectar datos reales
5. Más adelante sumar formularios más completos, validación y auth

## Dudas importantes que aparecieron hoy

- Formularios en `Next` y uso correcto de `label`, `htmlFor`, `id`, `name` y `type`
- Espaciado entre `label` e `input`
- Espaciado entre grupos de campos
- Cuándo envolver con `Fragment` (`<> </>`) y cuándo con `div`
- Cuándo usar `span`
- Si conviene usar componentes de `Typography` o texto simple
